
# How to run
  
    $ docker build -t go-docker .
    $ docker run -d -p 8081:8081 go-docker
    $ curl http://localhost:8081/
   
# API endpoints

  * `curl http://localhost:8081 `
  * `curl http://localhost:8081/forecast/stock `
    

## Requirements

  Docker - https://www.docker.com/get-started
